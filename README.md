# Dummy Car GPS/OBD Datalogger

Simple app to generate fake car data from onboard datalogger. Build for purpose testing backend api for GPS/OBD device.

## Tech Stack

***Python 3.10***

***Dependencies:*** 
* pytest
* request
* obd
* gps
* scipy 
* osrm-py
* rich
* tqdm
* numpy
* pandas

## Usage/Examples

Firstly rename ```./list.csv.sample``` to ```./list.csv```. Next fill it with desired data. Keep in mind that best syntax for location is ```Country, City, Street BuildingNumber``` e.g. ```Poland, Kielce, Zakladowa 1```
